# Password Manager
Projekt grupowy – menedżer haseł z analizą siły haseł i wykrywaniem wycieków.

## Funkcje:
- przechowywanie haseł
- analiza siły hasła
- sprawdzanie wycieków (np. API haveibeenpwned)
- logowanie użytkownika

## Technologie:
- Node.js (backend)
- React (frontend)
- MongoDB / SQLite
